To deploy the application:

1. Make a .env File in the root dir

Paste the following

PORT=3000
DEBUG=\*
MONGO_URL=localhost:27017/stock
API_LIST=['company','stocksintime','tickersearch']

2. Install mongo and load db dump

3. npm install
4. npm start
